# Interactive-Webpage

---

## Overview
This single‑page site asks for your name and age, then stores the data in your browser’s localStorage. Every time you return, it greets you personally, tells you your age in months, checks if you’re over 18, and shows you a motivational quote – five times, because you deserve the reminder.

# ✨ Features

| Feature | Description |
|---------|-------------|
| **Persistent Storage** | Name and age are saved in `localStorage` – no database needed. |
| **Personalised Greeting** | “Hello, [name]!” appears as soon as you submit the form or revisit the page. |
| **Age in Months** | Your age multiplied by 12 – a fun little calculation. |
| **Adult Content Check** | A friendly message tells you whether you’re old enough for adult content (if/else logic). |
| **Motivational Quote ×5** | The same quote appears five times, generated by a simple `for` loop. |
| **Dark, Luxurious Theme** | Eternal black background, marble white text, and warm gold accents. |
